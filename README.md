# MagicSnackBar
SnackBar creator module for Android




Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.IsMMA:MagicSnackBar:1.0.2'
	}
